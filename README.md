# UART Communication Protocol Simulator

## Overview

A modular UART communication protocol simulator written in C that demonstrates how UART frames are generated, transmitted, buffered, validated, and received.

The simulator models:

- UART frame generation
- Configurable UART settings
- Circular ring buffer
- Parity generation
- Checksum verification
- Error injection
- Buffer overflow handling
- Transmission statistics

This project is intended for learning embedded systems concepts and demonstrating firmware development skills.
